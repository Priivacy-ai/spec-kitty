## Objective (FR-001, FR-002, FR-003 ¬∑ finding I-1, I-9 ‚Äî HIGH, merge blocker)

## Context (settled design ‚Äî read `research.md` R1 + `contracts/doctor-doctrine-health.contract.md`)
Profile loading is **eager and all-or-nothing**: `repository._load()/list_all()` raises mid-iteration on `InlineReferenceRejectedError` (raised at `repository.py:~426`), aborting the whole layer load. `_collect_profile_health` (`doctor.py:~1955`) then swallows it via a broad `except Exception` and returns an **empty** report ‚Üí `DoctrineHealthReport.healthy == all([]) == True` (`_doctrine_health.py:~112`). A consumer-only catch was reviewed and **rejected** ‚Äî it cannot keep valid siblings visible. The fix is at the **load layer**.

## Subtasks
### T001 ‚Äî Load-layer skip + structural `healthy`-honesty + RC=1 (close the fail-to-green CLASS, not one mask)
The survey (debbie H2, proven) showed I-1 is one mask of a structural defect: `DoctrineHealthReport.healthy = all(packs)` is **vacuously `True` on empty** and **ignores `org_drg["errors"]`**, so several paths report green-when-broken. Make the whole class loud:
(a) **Load-layer skip** ‚Äî in `repository._load_layer` (per-file loop) catch `InlineReferenceRejectedError` ‚Üí `_record_skip` a `SkippedProfile{layer, path, profile_id, error_summary}`; the YAML is in hand so populate `profile_id` (the exception lacks it ‚Äî DD-2) and a **clear, readable** `error_summary` (forbidden field + migration hint). Valid siblings MUST keep loading.
(b) **Honest `healthy` flag** (`_doctrine_health.py`, ~line 112): `DoctrineHealthReport.healthy` ‚Üí `bool(self.packs) and all(...) and not self.org_drg.get("errors")` ‚Äî kills the vacuous `all([])==True` AND the org-DRG-error blind spot.
(c) **Collector records its crash** ‚Äî `_collect_profile_health` (`doctor.py:~1955`) must record the load error (e.g. into `org_drg["errors"]` / a `profile_load_error`) instead of degrading to an empty (green) report; "collector crashed" must be distinguishable from "genuinely zero profiles."
(d) **Human renderer default** (`doctor.py:~1892-1894`): a present pack with no/partial `pack_health` renders **degraded**, not green (`.get("healthy", False)`).
(e) **RC=1 on unhealthy** ‚Äî `doctor doctrine` currently always `raise typer.Exit(0)`; change to `raise typer.Exit(0 if report.healthy else 1)`. Loud-fail-with-traceability over hidden-fail (operator directive). The honest flag from (b) now drives the exit code.
