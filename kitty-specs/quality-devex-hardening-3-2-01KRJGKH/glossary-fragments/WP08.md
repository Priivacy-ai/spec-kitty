# WP08 glossary fragment

WP08 introduces no new canonical terms; it reinforces the **fail-safe default**
behavior in the stale-lane auto-rebase classifier per ADR
`architecture/2.x/adr/2026-05-14-1-stale-lane-auto-rebase-classifier-policy.md`.

The classifier rules and resolution surface (`Auto`, `Manual`,
`ConflictClassification`, `AutoRebaseReport`) are implementation entities,
not user-facing glossary terms. Any operator-facing language for the merge
flow (e.g. "stale lane", "auto-rebase") is already covered by the existing
glossary entries in `.kittify/glossaries/spec_kitty_core.yaml`.
