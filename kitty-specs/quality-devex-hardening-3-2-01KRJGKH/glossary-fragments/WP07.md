# Glossary Fragment — WP07

## Sonar quality gate

**Canonical term**: Sonar quality gate

**Definition**: The SonarCloud project-level pass/fail evaluation for
`Priivacy-ai_spec-kitty` on the new-code window. Aggregates six conditions
(reliability rating, security rating, maintainability rating, new coverage,
new duplicated lines density, new security hotspots reviewed). Reports `OK`
or `ERROR` per condition plus a top-level status of `OK` or `ERROR`.
Pulled via the public REST API at
`https://sonarcloud.io/api/qualitygates/project_status`; the helper
`work/snippets/sonarcloud_branch_review.sh` formats this for terminal use.

**Confidence**: 0.95

**Status**: active

**Source**: SonarCloud REST API (`qualitygates/project_status`); Spec Kitty
mission `quality-devex-hardening-3-2-01KRJGKH` FR-004, FR-013.

| `new_security_rating` | GT | 1 (A) |
| `new_maintainability_rating` | GT | 1 (A) |
| `new_coverage` | LT | 80% |
| `new_duplicated_lines_density` | GT | 3% |
| `new_security_hotspots_reviewed` | LT | 100% |

**Related terms**: Sonar hotspot, new-code window, SonarCloud, quality gate condition.
