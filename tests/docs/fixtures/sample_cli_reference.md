# CLI Command Reference

## spec-kitty foo

_Run the foo command_

```
Usage: spec-kitty foo [OPTIONS]
```

## spec-kitty bar

_Bar subcommand_

```
Usage: spec-kitty bar [OPTIONS]
```

## spec-kitty bar baz

_Baz of bar_

```
Usage: spec-kitty bar baz [OPTIONS]
```

## spec-kitty legacy-cmd

> **Deprecated**: legacy-cmd will be removed in 3.3

```
Usage: spec-kitty legacy-cmd [OPTIONS]
```

## spec-kitty tracker

_Tracker root_

```
Usage: spec-kitty tracker [OPTIONS]
```

## spec-kitty tracker list

_List tracker items_

```
Usage: spec-kitty tracker list [OPTIONS]
```
